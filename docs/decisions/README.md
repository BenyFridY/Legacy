# Decision Records (ADRs)

Registros de decisões de arquitetura — o **fio da "progressão de raciocínio"** que o case pede ver nos commits.

Cada ADR captura: o **contexto** (por que a decisão apareceu), a **decisão**, a **justificativa** (com evidências), as **alternativas rejeitadas** e as **consequências** (o que se ganha e o que se sacrifica).
